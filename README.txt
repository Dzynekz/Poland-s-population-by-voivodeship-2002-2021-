EN:
In the folder named "EN", there are items used in my work.
Among them, you can find my work on the population in Poland,
a CSV file containing the used data, a panel created in the Power BI program,
and a picture of the panel itself, which allows you to check
the appearance of the panel without downloading it.

Python file:
The file is used to connect to the GUS API and download population data.
It contains code that sends requests to the API and retrieves the relevant information.
Once the data is downloaded, the code processes it and saves it in a csv file. 
This allows the program to access up-to-date information about the population and use it to perform various tasks,
such as generating reports or analyzing trends.
The code presented in the project could have been written in a way that requires fewer API requests
and allows for faster data retrieval. However, it wasn't a priority for me, and I'm aware that there's room for improvement.

Plik Python:
Plik służy do połączenia z API GUS i pobrania danych na temat populacji.
Zawiera kod, który wysyła żądania do API i pobiera odpowiednie informacje.
Po pobraniu danych, kod przetwarza je i zapisuje w pliku CSV.
Pozwala to programowi uzyskać aktualne informacje o populacji i używać
ich do wykonywania różnych zadań, takich jak generowanie raportów
lub analizowanie trendów. Kod przedstawiony w projekcie mógłby zostać napisany
w taki sposób, aby wymagał mniejszej ilości zapytań do API i pozwolił na szybsze
pobieranie danych. Jednakże, nie była to dla mnie priorytetowa kwestia i jestem świadomy,
że istnieje możliwość ulepszenia kodu.
